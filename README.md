오늘 하루의 ⭐️갓생 인증⭐️하는 서비스입니다.

## 주제 선정:
저는 친구들과 하루 1시간 공부 인증글을 올리는 스터디를 2023년 부터 해왔습니다.
카톡방에 사진을 올리는 것으로 인증했는데, 내꺼만 따로 모아서 보고 싶다 라는 생각으로 아이디어를 떠올렸습니다.

우테코 디스코드에서 많은 사람들이 1일 1솔, 오운완과 같은 챌린지를 파서 하루의 인증을 남기는 것을 보고
자유롭게 나의 성장을 기록하는 서비스를 만들기로 결정했습니다.

더 자세한 것은 게시물 참고해주세요  ➡️ [[우테코] 오픈 미션 4주차 - 개발 홀로서기 도전 velog](https://velog.io/@wldud8297/%EC%9A%B0%ED%85%8C%EC%BD%94-%EC%98%A4%ED%94%88-%EB%AF%B8%EC%85%98-%EA%B0%9C%EB%B0%9C-%ED%99%80%EB%A1%9C%EC%84%9C%EA%B8%B0-%EB%8F%84%EC%A0%84)

(개발일지도 해당 블로그로 정리해서 추후 업로드 예정입니다!)

## 서비스 소개

### 1. 로그인

아직 카카오톡 로그인이 불가능합니다❗️네이버 로그인으로 부탁드립니다!

![](https://velog.velcdn.com/images/wldud8297/post/dcf87096-97dd-4b76-a837-185582dbf363/image.png)

### 2. 챌린지 추가
메인 화면에서 챌린지 찾기를 들어가면 원하는 챌린지를 선택할 수 있습니다 😄

![](https://velog.velcdn.com/images/wldud8297/post/36559534-1d91-493e-b8b5-b935d574a0e5/image.png)
 
하트를 누르면 다시 챌린지에서 탈퇴할 수 있습니다.

<img src="https://velog.velcdn.com/images/wldud8297/post/2b9a86f2-6bc2-40fd-a4df-a7589a849c1a/image.png" width="60%">

### 3. 챌린지 참여
메인 화면에서 카메라 버튼을 누르면 오늘의 인증을 할 수 있습니다.  
오늘의 인증을 완료하면 카메라가 ✅로 변합니다.

- 한 챌린지당 하루에 1개 올릴 수 있습니다.
- 사진은 최대 3개 올릴 수 있습니다.
- 나만 보기(비공개)로 공유할 경우 다른 사람들은 볼 수 있습니다.

![](https://velog.velcdn.com/images/wldud8297/post/96a04747-6ee3-4875-b9de-ae6fcd7255e7/image.png)

체크 표시를 누르면 오늘의 인증 기록을 삭제할 수 있습니다

<img src="https://velog.velcdn.com/images/wldud8297/post/1a6a797b-c24e-4b83-a1d3-3728867e4453/image.png" width="60%">


### 3. 챌린지 인증방
메인 화면에서 챌린지를 누르면 다른 사람들과 공유된 챌린지 인증글을 확인할 수 있습니다.   
- 공개와 나만 보기 옵션을 선택할 수 있습니다.
- 공개는 공개로 포스트한 모든 게시물이 표시됩니다.
- 나만 보기는 내가 올린 인증 게시물만 표시됩니다. 비공개는 위에 "🔒 비공개"가 표시됩니다.

<img src="https://velog.velcdn.com/images/wldud8297/post/375a508b-bbb4-4c9f-8b3f-07612ba2719b/image.png" >


### 4. 마이페이지

메인 화면에서 프로필을 누르면 나의 정보 조회 및 수정을 할 수 있습니다.  

- 생년월일, 이메일 주소는 수정 불가합니다!
